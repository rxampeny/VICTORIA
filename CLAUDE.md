# VICTORIA - Aniversaris Víctor Català 1971

## Descripción
App web para gestionar cumpleaños del grupo "Víctor Català 1971".
Calendario de cumpleaños con notificaciones por SMS y email.

## Stack Tecnológico
- Frontend: HTML, CSS, JavaScript vanilla
- Backend: Google Apps Script (API REST)
- Base de datos: Google Sheets
- SMS: Twilio API
- Hosting: Netlify
- URL: https://victor-catala-19711971.netlify.app/
